What is Stream?

Stream is a Plugin creation service for Minecraft Servers. If you want a plugin, we can make it! Once Sponge is officially out, we will offer support for Sponge plugins.

What can we do?

We offer a wide arrange of plugin choices, including Minigames, MySQL, Packets, NMS, and much more. We also code plugins for Spigot & Glowstone, which both use the Bukkit API.
